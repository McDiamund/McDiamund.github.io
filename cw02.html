Exercise 6

HTML Tags Used For School Website

DIV - contains sections to seperate information. You could use divs to seperate student information personal to each person and then another section for assignments, exams, etc.

Table - can be used to lay out a schedule for each exam and times for each class, etc.

h1 - can be used to title pages and display important imformation to give context to the information. Could also be used to the display the students name and information and change depending on the account of course.

h3 - can be used to give deeper compartmentlization of sections. for example classes, subjects, topics.

h2 - same as h3, use to compartmentlize imformation

p - the tag used to display large pieces of information, important

hr - can be used as a visual aid in seperating sections

br - used to control the size of information present in p contents, always needed

tr - makes rows for tables and can be seperated further to make sections for displayed data. For example class schedule, test times, etc.

th - allows visually a place to assign what to put in tables, can be used for things like subject or classes or anything you could put in a graph form.

td - the block or cell in a table that pieces of data actually goes like times, test, etc.
